- [Understanding services](https://docs.ros.org/en/foxy/Tutorials/Beginner-CLI-Tools/Understanding-ROS2-Services/Understanding-ROS2-Services.html)

目标:
- 能够通过`ros2 service list -t`命令查看系统中当前所有活动服务的列表和服务类型。
- 能够通过`ros2 service type <service_name>`命令查看某个服务(例如 /clear)的服务类型。
- 能够通过`ros2 service find <type_name>`命令查看某个服务类型(例如 std_srvs/srv/Empty)的服务。
- 能够通过`ros2 interface show <type_name>.srv`命令查看某个服务类型(例如 turtlesim/srv/Spawn)的接口签名信息。
- 能够通过`ros2 service call <service_name> <service_type> <arguments>`命令调用某个服务。

命令行调用服务示例。

清除窗口中绘制的所有线条:
```s
ros2 service call /clear std_srvs/srv/Empty
```

在指定位置生成一个新的海龟:
```s
ros2 service call /spawn turtlesim/srv/Spawn "{x: 2, y: 2, theta: 0.2, name: ''}"
```
