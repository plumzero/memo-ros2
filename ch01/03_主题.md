
参考:
- [Understanding topics](https://docs.ros.org/en/foxy/Tutorials/Beginner-CLI-Tools/Understanding-ROS2-Topics/Understanding-ROS2-Topics.html)

目标:
- 了解 rqt_graph 的使用，能够通过该工具查看不断变化的节点和主题，以及它们之间的连接。
- 能够通过`ros2 topic list`命令查看所有主题列表。
- 能够通过`ros2 topic echo <topic_name>`命令查看某个主题(例如 /turtle1/cmd_vel)的回显数据。
- 能够通过`ros2 topic info <topic_name>`命令查看某个主题(例如 /turtle1/cmd_vel)的信息(例如消息类型、发布者和订阅者数量)。
- 能够通过`ros2 interface show <msg_type>`命令查看某个消息类型(例如 geometry_msgs/msg/Twist)的数据结构。
- 能够通过`ros2 topic pub <topic_name> <msg_type> '<args>'`命令控制海龟的移动。
- 能够通过`ros2 topic hz <topic_name>`命令查看节点向主题(例如 /turtle1/pose)发布数据的速率。

命令行控制海龟移动示例。

一条控制:
```s
ros2 topic pub --once /turtle1/cmd_vel geometry_msgs/msg/Twist "{linear: {x: 2.0, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 1.8}}"
```

频率控制:
```s
ros2 topic pub --rate 1 /turtle1/cmd_vel geometry_msgs/msg/Twist "{linear: {x: 2.0, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 1.8}}"
```
