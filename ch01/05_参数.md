
参考:
- [Understanding parameters](https://docs.ros.org/en/foxy/Tutorials/Beginner-CLI-Tools/Understanding-ROS2-Parameters/Understanding-ROS2-Parameters.html)

目标:
- 能够通过`ros2 param list`命令查看当前活动所有节点的参数。
- 能够通过`ros2 param get <node_name> <parameter_name>`命令查看某个节点下某个参数的类型和当前值。
- 能够通过`ros2 param set <node_name> <parameter_name> <value>`命令更改某个节点下的某个参数值。
- 能够通过`ros2 param dump <node_name>`命令将节点的所有当前参数值保存到文件中。
- 能够通过`ros2 param load <node_name> <parameter_file>`命令将参数文件内容加载到节点的参数中。
- 能够通过`ros2 run <package_name> <executable_name> --ros-args --params-file <file_name>`命令在启动时加载参数文件。
