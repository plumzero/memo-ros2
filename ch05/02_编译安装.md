
### 版本说明

- 版本为[v3.4.1](https://github.com/eProsima/Fast-DDS/tree/v3.4.1)。
- cmake 版本要求 3.20 或以上。

### 依赖安装

下面的库通过 Linux 包管理方式安装时，版本可能不符合要求，所以采用编译后安装。其他的依赖库(如 boost, openssl 等)可采用包管理安装方式。

[Fast-CDR:v2.3.4](https://github.com/eProsima/Fast-CDR/tree/v2.3.4)克隆后，进入顶层目录下执行如下命令:
```s
mkdir build && cd build
cmake .. -DCMAKE_INSTALL_PREFIX=~/fastdds-dependency/fastcdr
make -j4
make install
```

[Asio:v1.13.0](https://github.com/chriskohlhoff/asio/tree/asio-1-13-0)克隆后，进入顶层目录下执行如下命令:
```s
cd asio
./autogen.sh
./configure --prefix ~/fastdds-dependency/asio
make -j4
make install
```

[foonathan_memory:v1.3.2](https://github.com/eProsima/foonathan_memory_vendor/tree/v1.3.2)克隆后，进入顶层目录下执行如下命令:
```s
mkdir build && cd build
cmake .. -DCMAKE_INSTALL_PREFIX=~/fastdds-dependency/foonathan_memory
make -j4
make install
```

### 编译安装

```s
export CMAKE_PREFIX_PATH=~/fastdds-dependency/fastcdr:~/fastdds-dependency/asio/include:~/fastdds-dependency/foonathan_memory
cmake .. -DCMAKE_INSTALL_PREFIX=~/fastdds
make -j4
make install
```
